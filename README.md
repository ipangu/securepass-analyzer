**SecurePass Analyzer** 是一个开源密码强度检测工具，帮助用户检查密码是否安全。支持 **网页版、命令行工具（CLI）和 API**，适用于开发者和安全研究人员。

## 🚀 功能
✅ **密码强度分析**（长度、常见密码、复杂度）  
✅ **网页版交互界面**（实时检测）  
✅ **命令行工具（CLI）**（适合开发者集成）  
✅ **REST API**（可嵌入其他应用）  
✅ **自动化测试 & CI/CD**（GitHub Actions）
